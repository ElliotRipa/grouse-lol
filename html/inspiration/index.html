sadgrl.online

This image is a good reference for an attractive pixel style. It uses the classical gradient style where, instead of gradually changing the hex values of all the pixels gradually, instead it starts to interlace the rows with occasional pixels of the upcoming colour. This is rather reminiscent of the style often used in Windows 95, or even XP.
Some elements of note are the aforementioned background, which has been backed up here; the topbar, which has been backed up here; and the font Freepixel.ttf, which is downloadable here.
